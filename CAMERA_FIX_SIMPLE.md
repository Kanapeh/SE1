# 📷 حل آسان مشکل دوربین

## ❌ مشکل: 
دوربین در تماس تصویری کار نمی‌کند

## ✅ راه‌حل آسان:

### مرحله ۱: بستن سرورهای فعلی
```bash
Ctrl + C
```
(در همه terminal هایی که Next.js اجرا است)

### مرحله ۲: اجرای سرور جدید
```bash
npm run dev:camera
```

### مرحله ۳: استفاده از آدرس مناسب

Script به شما چند آدرس می‌دهد:

#### 🏠 برای همین کامپیوتر:
```
http://localhost:3000/students/temp-user-id/video-call
```

#### 📱 برای گوشی یا تبلت (در همین WiFi):
```
http://192.168.x.x:3000/students/temp-user-id/video-call
```

---

## 🎯 چرا کار می‌کند؟

- **localhost**: مرورگر به دوربین دسترسی می‌دهد
- **IP محلی**: برای دستگاه‌های شبکه محلی مجاز است
- **بدون نیاز به ngrok**: خیلی ساده‌تر!

---

## 🧪 تست سریع:

1. `npm run dev:camera` اجرا کنید
2. آدرس localhost را در مرورگر باز کنید
3. به `/test-video` بروید
4. روی "Allow" برای دوربین کلیک کنید
5. باید خودتان را ببینید! ✨

---

## 🔧 اگر باز هم کار نکرد:

### گزینه ۱: تست مرورگر دیگر
- Chrome، Firefox، Safari یا Edge امتحان کنید

### گزینه ۲: بررسی دسترسی‌ها
- در صفحه تماس دکمه "راهنما" کلیک کنید
- مراحل تنظیم مرورگر را دنبال کنید

### گزینه ۳: HTTPS واقعی (اگر لازم باشد)
```bash
# نصب localtunnel (آسان‌تر از ngrok)
npm install -g localtunnel

# Terminal 1
npm run dev

# Terminal 2  
npx localtunnel --port 3000
```
