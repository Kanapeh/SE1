# 🚨 حل فوری مشکل دوربین

## مشکل شما:
دوربین در تماس تصویری کار نمی‌کند چون سایت روی **HTTP** اجرا می‌شود.

---

## ✅ حل فوری:

### مرحله ۱: بستن سرور فعلی
```bash
Ctrl + C
```

### مرحله ۲: اجرای سرور HTTPS
```bash
npm run dev:https
```

### مرحله ۳: استفاده از آدرس HTTPS
سرور به شما آدرسی مثل این می‌دهد:
```
https://abc123.ngrok.io
```

### مرحله ۴: تست دوربین
به این آدرس بروید:
```
https://abc123.ngrok.io/students/temp-user-id/video-call
```

---

## 🎯 اگر script کار نکرد:

### راه دستی:
```bash
# Terminal 1: اجرای Next.js
npm run dev

# Terminal 2: اجرای ngrok
npx ngrok http 3000
```

سپس از آدرس HTTPS که ngrok می‌دهد استفاده کنید.

---

## ✨ تست سریع:

1. آدرس HTTPS را در مرورگر باز کنید
2. `/test-video` اضافه کنید
3. روی "Allow" برای دوربین کلیک کنید
4. باید تصویر خودتان را ببینید!

---

## 📱 مشکل دارید؟

- مطمئن شوید آدرس با `https://` شروع می‌شود
- F5 کنید (refresh page)
- در صفحه تماس، دکمه "راهنما" کلیک کنید
