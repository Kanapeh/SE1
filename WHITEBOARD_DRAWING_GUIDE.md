## 🎯 **ویژگی‌های فنی:**

### ✅ **Canvas API:**
```javascript
// رسم با خط نرم
context.lineCap = 'round';
context.lineJoin = 'round';

// پاک‌کن با شفافیت
context.globalCompositeOperation = 'destination-out';

// ذخیره تاریخچه
setDrawingHistory(prev => [...prev, imageData]);
```

### ✅ **Event Handling:**
```javascript
onMouseDown: شروع رسم
onMouseMove: رسم در حین حرکت
onMouseUp: پایان رسم
onMouseLeave: توقف رسم
```

### ✅ **State Management:**
```javascript
drawingHistory: تاریخچه تمام تغییرات
historyIndex: موقعیت فعلی در تاریخچه
isDrawing: وضعیت رسم
drawingTool: ابزار انتخاب شده
drawingColor: رنگ انتخاب شده
```
